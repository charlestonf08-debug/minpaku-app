# 民泊運営管理システム

かわけん・りょうこ の共同民泊運営（山王・玉津・橘）を一元管理するWebアプリです。

## セットアップ手順

### 1. Supabaseプロジェクト作成

1. [supabase.com](https://supabase.com) にアクセスしてアカウント作成・ログイン
2. 「New project」でプロジェクト作成
3. Project Settings → API から以下を控える
   - **Project URL** (例: `https://xxxxx.supabase.co`)
   - **anon public key** (例: `eyJhbGci...`)

### 2. データベース初期化

1. Supabase Dashboard → SQL Editor を開く
2. `supabase-setup.sql` の内容を貼り付けて実行
3. `properties`テーブルに山王・玉津・橘が追加される

### 3. ユーザー作成

1. Supabase Dashboard → Authentication → Users
2. 「Add user」でかわけん・りょうこ のアカウントを作成
   - メールアドレスとパスワードを設定

### 4. アプリ起動

`index.html` をブラウザで開く（ローカルサーバー推奨）：

```bash
# Python がある場合
python3 -m http.server 3456
# → http://localhost:3456 にアクセス
```

### 5. 初回設定

1. Supabase URL と Anon Key を入力して「接続して開始」
2. 作成したアカウントでログイン
3. 設定 → スタッフ → 外部スタッフを追加
4. 設定 → 交通費 → 各スタッフの物件別交通費を設定

## 機能一覧

| ページ | 内容 |
|--------|------|
| 精算サマリー | 月次精算計算・振込金額の確認 |
| シフト管理 | 清掃シフトの登録・完了処理（カレンダー/リスト） |
| 収支入力 | 売上・固定費・変動費の入力 |
| レポート | 月別推移グラフ・スタッフ集計 |
| 設定 | スタッフ管理・交通費設定 |

## 精算ロジック

```
全売上 = 山王売上 + 玉津売上
全費用 = 固定費 + 外部清掃費・交通費 + 変動費(両者) + 清掃報酬(両者)
利益 = 全売上 - 全費用
各自の取り分 = 利益 ÷ 2

かわけんの実際の手元 = 山王売上 + かわけん清掃報酬 - 固定費 - 外部清掃費 - かわけん変動費
精算額 = かわけんの実際の手元 - 各自の取り分

精算額 > 0 → かわけん → りょうこ に振込
精算額 < 0 → りょうこ → かわけん に振込
```
